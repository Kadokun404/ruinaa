log_001 — identitas

Siapa Ruina?

Ruina adalah inti dari sistem bot Z-Hiruira's — bagian yang memutuskan kapan harus diam, kapan harus menjawab, dan kapan sebuah percakapan butuh sesuatu yang lebih dari respons template.

Dibangun untuk terasa seperti lawan bicara yang serba bisa, bukan mesin yang cuma menjalankan perintah — kini juga bisa kamu coba langsung lewat live chat di bawah.

log_004 — akses publik

Endpoint API

Dua endpoint publik, format response konsisten — sekali bikin parser, pakai di mana aja.

POST

/api/ruina

Kirim pesan, dapat balasan dari core Ruina.

curl -X POST https://ruina.z-hiruira.biz.id/api/ruina \
  -H "Content-Type: application/json" \
  -d '{ "message": "halo ruina" }'
{
  "ok": true,
  "author": "Z-Hiruira's",
  "result": { "reply": "Halo! Ada yang bisa dibantu?" },
  "meta": { "model": "ruina-core" }
}

/api/developer

Info publik developer di balik Z-Hiruira's — profil, skill, status.
